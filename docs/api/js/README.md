# JS Web3 SDK

虽然大多数现有的 SDK 可以开箱即用，但部署智能合约或使用独特的 zkSync 功能（例如以其他代币支付费用）需要在以太坊交易默认包含字段的基础上，提供额外的字段。

为了提供对 zkSync 2.0 所有功能的轻松访问，创建了 `zksync-web3` JavaScript SDK，它基于 `ethers` JavaScript 库。 本节作为其参考。
